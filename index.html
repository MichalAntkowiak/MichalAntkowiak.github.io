Projekty Mechatroniczne

Smart System Hydroponiczny (Project-Based Learning)

Projekt interdyscyplinarnego urządzenia smart-gardening realizowany w zespole akademickim. Układ automatyzuje procesy opieki nad roślinami hydroponicznymi poprzez zbieranie danych środowiskowych oraz aktywną reakcję układów wykonawczych.

Mój wkład: Opracowanie mechaniki urządzenia, w tym konstrukcji obrotowej ramy łożyskowanej kulkowo, umożliwiającej płynny ruch doniczki wokół osi w celu równomiernego naświetlania rośliny. Integracja czujników wilgotności podłoża i temperatury z mikrokontrolerem oraz oprogramowanie logiki sterowania pompą hydrauliczną i napędem w MicroPythonie.

Aplikacja desktopowa napisana w języku C++ z interfejsem graficznym w bibliotece Qt. Implementuje pełną logikę gry w szachy w paradygmacie obiektowym oraz moduł bota szachowego wykonującego analizę i ewaluację pozycji.

bool Board::validateMove(int fromX, int fromY, int toX, int toY) {
    Piece* p = grid[fromY][fromX];
    if (p == nullptr) return false;
    
    if (!p->isValidPattern(fromX, fromY, toX, toY)) return false;
    return isPathClear(fromX, fromY, toX, toY);
}
